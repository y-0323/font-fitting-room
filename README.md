# フォント試着室 — Font Fitting Room

見出し×本文のフォントの組み合わせを、実際の文章で横並び比較できるツールです。
**無料フォント（Google Fonts 約1900書体）**だけを扱うので、誰でもそのまま使えて、商用利用もOKです。

## 使い方

`index.html` をブラウザで開くだけ。サーバー不要です。

- 見出し / 本文の**フォント名をクリック**→ 検索して選択（和文・欧文どちらも）
- **詳細を調整**でサイズ・太さ・行間・字間を変更
- **CSSをコピー**：`@import` 付きCSSをコピー → 貼ればすぐ使えます
- **画像で保存**：カードをPNGで書き出し
- **共有リンク**：今の組み合わせをURL化。**誰の環境でも同じ見た目**で開けます
- **Light / Dark**：背景を切り替えて見え方を確認
- **おまかせ**：相性の良い組み合わせをランダム提案

## なぜ無料フォントだけ？

- **全部無料・オープンライセンス**（商用利用OK）で、使う人もすぐ導入できる
- Webフォントとして配信されるので、**共有リンクや実装したサイトで、相手の環境にもそのまま表示される**（モリサワ等の有料/ローカルフォントは相手の環境に無いと表示されない）

## ちょっとした注意

- **欧文専用フォント**（Playfair Display など）を日本語テキストに当てると、日本語の字を持たないため別書体で代替表示されます。その場合はカードに注意が出ます。和文を比較したいときはピッカーの「和文（日本語対応）」グループから選んでください。
- 表示には Google Fonts の読み込みが必要なため、**オフラインでは表示できません**（その場合は端末標準フォントで代替）。
